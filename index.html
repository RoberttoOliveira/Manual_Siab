Bem Vindo ao Manual Siab Facil

Este manual foi desenvolvido para facilitar o entendimento aos usuários do SIAB FÁCIL sobre todas as telas e rotinas do sistema. Para facilitar a navegação neste manual, utilize os botões acima para Ocultar e Mostrar o índice, Voltar à página anterior ou posterior e Imprimir.

Para saber sobre todas as alterações realizadas tanto na versão Desktop quanto na versão Móvel do sistema, basta clicar nos respectivos botões Notas da Versão. As versões estão documentadas em ordem decrescente, assim a primeira página refere-se às implementações realizadas na última versão. Aconselhamos que leia este documento a cada nova atualização.
